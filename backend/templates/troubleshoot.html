Troubleshooting Center

Common deployment, infrastructure, networking, Kubernetes and Terraform issues with solutions.

Terraform Init Failed

Cause: Terraform modules or providers are not initialized properly.

Fix: Run terraform init before terraform apply.

AWS Credentials Invalid

Cause: Incorrect AWS Access Key or Secret Key.

Fix: Configure AWS CLI again using aws configure.

Kubernetes Pods CrashLoopBackOff

Cause: Container startup failure or invalid image.

Fix: Check pod logs using kubectl logs.

Docker Build Failed

Cause: Dockerfile syntax error or missing dependencies.

Fix: Validate Dockerfile and rebuild image.

CI/CD Pipeline Failed

Cause: Build/test stage failure.

Fix: Review pipeline logs and dependency versions.

GitHub Repository Not Detected

Cause: Invalid repository URL or private repo access denied.

Fix: Verify repository URL and authentication.

Application Not Accessible

Cause: Security Group or Load Balancer issue.

Fix: Open ports 80 and 443 in AWS Security Groups.

VPC/Subnet Misconfiguration

Cause: Missing route tables or internet gateway.

Fix: Configure proper subnet routing and attach IGW.

Monitoring Dashboard Not Loading

Cause: Grafana or Prometheus service stopped.

Fix: Restart monitoring containers/services.

Terraform State Lock Error

Cause: State file locked due to interrupted deployment.

Fix: Release state lock carefully before redeployment.
